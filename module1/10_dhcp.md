# DHCP Address Assignment

# What is DHCP?

DHCP stands for Dynamic Host Configuration Protocol.

It automatically assigns network configuration such as:

- IP address
- Subnet mask
- Default gateway
- DNS server

# DHCP Process

The DHCP process follows four steps called DORA.

1. Discover
2. Offer
3. Request
4. Acknowledge

# Process Explanation

1. Client sends DHCP Discover message.
2. Server replies with DHCP Offer.
3. Client sends DHCP Request.
4. Server sends DHCP Acknowledge.
